Connect to Cloud-Bridge

Link your own Spotify developer app to Cloud-Bridge. This page supports an independent educational AAOS template and is not affiliated with Spotify.

Bring your own Spotify developer app:

  1. Go to the Spotify Developer Dashboard and log in.
  2. Click Create App (or open an existing one).
  3. In the app Settings, ensure your Redirect URI is set exactly to:
    https://wasidremin.github.io/AAOS_Spotify_Cloud_Bridge/
  4. Copy your Client ID and Client Secret below. The historical GitHub Pages URL stays the same for redirect compatibility.
